# Manan Negi — GenAI & Automation Engineer Portfolio

A premium, high-performance personal portfolio showcasing GenAI-powered catalog automations, full-stack data dashboards, and ETL pipelines. Built with vanilla HTML, CSS, JavaScript, and powered by **GSAP (GreenSock)** for smooth scrollytelling visual transitions.

---

## ✨ Features & Highlights

*   **Split-Screen Scrollytelling**: An ESPN-inspired projects layout showing visual mockups inside macOS browser frames on the left, synchronized to scrollable content cards on the right.
*   **GSAP & ScrollTrigger Animations**: Hardware-accelerated staggered entrance animations on section headings, journey story cards, and the skills bento grid.
*   **Responsive Details Modal**: Clickable mockup overlays with blurred background frames and a responsive image carousel showing project detail screenshots without letterboxing.
*   **Optimized Custom Cursor**: A smooth custom cursor follower with mouse-delegate hover states. It automatically sleeps when the cursor stops moving to preserve system CPU and GPU cycles.
*   **Fully Responsive**: Reverts to unified single-column vertical cards on mobile and tablet viewports.

## ⚙️ Local Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/MNegi17/Manan-Portfolio.git
   ```
2. Open the directory:
   ```bash
   cd Manan-Portfolio
   ```
3. Run a local server to view (e.g. Python):
   ```bash
   python -m http.server 8000
   ```
4. Navigate to `http://localhost:8000` in your browser.
